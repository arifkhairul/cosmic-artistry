## Setup
Download [Node.js](https://nodejs.org/). Run this followed commands:
``` bash
# Install dependencies (only for first time) | [node_modules] folder
npm install

# Run the local server at localhost:5173
npm run dev

# Build for production | [dist] folder
npm run build
```
